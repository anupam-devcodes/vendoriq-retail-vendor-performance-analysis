# VendorIQ: Retail Vendor Performance & Profitability Intelligence

VendorIQ is an end-to-end data analytics project focused on vendor performance analysis in the retail and wholesale industry.

The goal of this project is to help a business optimize profitability, inventory efficiency, purchasing decisions, and vendor relationships using SQL, Python, statistical analysis, and Power BI.

## Business Problem

Retail and wholesale companies need to manage inventory and vendor relationships effectively to avoid losses caused by poor pricing, slow inventory turnover, low-margin products, and overdependence on specific vendors.

This project aims to:

- Identify underperforming brands that may need pricing or promotional adjustments.
- Determine top vendors contributing to sales and gross profit.
- Analyze the impact of bulk purchasing on unit costs.
- Assess inventory turnover to reduce holding costs.
- Compare profitability between high-performing and low-performing vendors.
